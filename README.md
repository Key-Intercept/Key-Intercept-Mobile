Welcome to the long awaited mobile edition of Key-Intercept a plugin for discord which helps you talk ~~un~~properly.

For the full description, usage guide, wiki, etc... Please click [Right Here](https://github.com/Key-Intercept/key-intercept?tab=readme-ov-file#key-intercept)

Please also go there to log bugs.

## Install

> Note: This relies on Aliucord, which (like Vencord) is an unsupported use of Discord, which breaks the TOS to use.

> This only supports Android, quite frankly Apple can go eff itself.

### Aliucord Install
First install [Aliucord](https://github.com/Aliucord/Aliucord#-installation), if you already have aliucord installed then skip to [Plugin Install](#plugin-install)
1. Download and install the [Aliucord Manager](https://github.com/Aliucord/Manager/releases/latest)
2. Open Aliucord Manager
3. Grant permission requests
4. Click "New Install"
5. Click Install
> If this step fails, don't message me. Please send a message on the `#support` channel of the [Aliucord Discord Server](https://discord.gg/EsNDvBaHVU0)
6. Click Launch
7. Sign into discord


### Plugin Install
To install the plugin, you now have 3 methods, all of which are viable.

#### Option A
1. Go and find a discord message for `@supersliser' (the person who posted this in wto).
2. Hold down on the message (not the profile)
3. You should see an option that says "View supersliser's Plugins", click this.
4. You should see a blue button that says "Key-Intercept-Mobile". click this.

#### Option B
1. Find a message containing the link to this github repo.
2. Hold down on the message
3. You should see an option that says "Install Key-Intercept-Mobile", press this.

#### Option C (fallback method)
If the other 2 arent working, try this:
1. Go to the [Builds Branch](https://github.com/Key-Intercept/Key-Intercept-Mobile/tree/builds)
2. Click "KeyIntercept.zip" and press download
3. Move the file from the `Downloads` folder to the `Aliucord/plugins`
4. Close and reopen Aliucord
